# ⌨️ 键盘快捷键

BibleMate AI 提示栏位支援以下键盘快捷键：

- `Ctrl+Y`：说明资讯
- `Ctrl+S` 或 `Esc+ENTER` 或 `Alt+ENTER`：提交输入
- `Ctrl+N`：新对话
- `Esc+I`：汇入对话
- `Esc+O`：编辑对话
- `Ctrl+O`：在文字编辑器中编辑输入
- `Ctrl+Q`：退出输入
- `Ctrl+R`：重设输入
- `Ctrl+Z`：复原输入变更
- `Ctrl+W`：储存提示/计划
- `Esc+W`：删除提示/计划
- `Ctrl+L`：开启提示/计划
- `Esc+L`：搜寻提示/计划
- `Ctrl+B`：开启圣经相关功能
- `Ctrl+C`：开启圣经注释
- `Ctrl+V`：开启圣经经文功能
- `Ctrl+X`：开启串珠功能
- `Ctrl+F`：开启搜寻功能
- `Ctrl+J`：更改 AI 模式
- `Esc+T`：切换自动工具选择
- `Ctrl+G`：切换自动输入建议
- `Esc+G`：产生提示想法
- `Ctrl+P`：切换自动提示工程
- `Esc+P`：改进提示内容
- `Ctrl+D`：删除
- `Ctrl+H`：倒退键
- `Ctrl+W`：删除前一个单字
- `Ctrl+U`：删除文字直到行首
- `Ctrl+K`：删除文字直到行尾
- `Ctrl+A`：移至行首
- `Ctrl+E`：移至行尾
- `Ctrl+LEFT`：向左移动一个单字
- `Ctrl+RIGHT`：向右移动一个单字
- `Ctrl+UP`：向上卷动
- `Ctrl+DOWN`：向下卷动
- `Shift+TAB`：插入四个空格
- `TAB` 或 `Ctrl+I`：开启输入建议选单
- `Esc+Esc`：关闭输入建议选单

## 取消载入 AI 回应

在等待回应时，按 `Ctrl+C` 一两次直到载入取消。

## 终端机应用程式的常用快捷键：

这取决于您的终端机应用程式设定：

- `Ctrl+Shift+C`：复制
- `Ctrl+Shift+V`：贴上
