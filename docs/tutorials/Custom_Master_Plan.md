# 📝 自订主要计划

在某些情况下，您可能希望为复杂的任务指定一个包含多个工具的「自订计划」，而不是在全自动代理模式下自动生成「主要计划」。

您可以使用自己的自订「主要计划」，而不是由 BibleMate AI 生成的计划。为此，请在您的 BibleMate AI 提示中以「@@」开头，后面跟著您自己的圣经学习主要计划。

例如：

```
@@ 分析约翰福音 3:16，步骤如下：
1. 呼叫工具「retrieve_english_bible_verses」以获取圣经文本，
2. 呼叫工具「retrieve_bible_cross_references」以获取圣经串珠，
3. 呼叫工具「interpret_new_testament_verse」以进行解释，以及
4. 呼叫工具「write_bible_theology」以解释其神学。
```

观看此影片：https://youtu.be/Lejq0sAx030

另一个例子：

```
@@ 撰写申命记 6:4 的感人灵修，步骤如下：
1. 使用 @study_old_testament_themes 分析主题
2. 使用 @identify_bible_keywords 识别并解释经文中的关键圣经词汇
3. 使用 @write_bible_insights 撰写灵修见解
4. 使用 @write_bible_applications 将经文与日常生活联系起来
5. 使用 @write_bible_devotion 撰写感人灵修。
确保每个步骤都清晰明确，最终输出连贯且鼓舞人心
```

观看此影片：https://youtu.be/NYPyujFG09E

这「@@」技巧可应用在不同的 AI 模式上。

## 储存常用自订计划

在 BibleMate AI 中编写自订计划后，按下 `Ctrl+W` 即可储存。

按下 `Esc+W` 即可删除已储存的计划。

按下 `Ctrl+L` 即可开启计划。

按下 `Esc+L` 即可搜寻计划。
