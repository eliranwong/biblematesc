# 🔧 单一工具选择

在某些情况下，您可能希望为简单的任务指定一个特定的工具，而不是在全自动 `agent 模式` 中自动选择多个工具。

您可以透过在提示开头以 `@` 加上工具名称来指定单一工具。例如：

```
@retrieve_bible_cross_references 申命记 6:4；约翰福音 3:16
```

观看此影片：https://youtu.be/50m1KRj6uhs

BibleMate AI 也可以自动选择单一工具来处理您的请求。只需在请求开头以 `@` 后面加上一个空格，而不是指定工具名称。例如：

```
@ 阅读申命记 6:4；约翰福音 3:16
```

## 自动工具选择

在 `agent` 和 `partner` 模式中，工具预设为自动选择。

在 `mode` 模式中，您可以输入 `.autotool` 或 `Esc+T` 来切换自动工具选择。启用此功能后，您无需手动在请求前加上 `@ `。
